# M3U8

`m3u8` 命令用于生成 `M3U8` 格式的视频播放地址。

```shell
pixelfs m3u8 <file>
```

- `file`: 指定要生成 M3U8 格式的视频文件。

**选项**

- `--bitrate`: 指定视频的码率。默认为原始码率，示例 `5000`。
- `--height`: 指定视频的高度。默认为原始高度，示例 `1080`。
- `--width`: 指定视频的宽度。默认为原始宽度，示例 `1920`。

## 示例

```shell
pixelfs m3u8 --bitrate 5000 --height 1080 --width 1920 video.mp4
```
